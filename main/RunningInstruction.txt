First chỉ đường dẫn tới thư mục file_cpp (là thư mục chứa các file cpp)

xong chạy trong terminal bằng lệnh

"
g++ *.cpp -o main
.\main.exe
"

*.cpp là thay thế cho tất cả các file có đuôi .cpp
Các đường dẫn include file header hay file input có dùng ../ 

ví dụ

../Header/Semester.h

Là lùi về 1 thư mục là trở về main rùi đi vô folder Header có file .hay
